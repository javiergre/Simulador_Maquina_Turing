# Simulador de Máquina de Turing

## Descripción
Simulador interactivo de Máquina de Turing desarrollado en Python con interfaz gráfica. Permite visualizar el funcionamiento paso a paso de una MT y evaluar cadenas contra 10 expresiones regulares predefinidas.

## Características
- Interfaz gráfica moderna tipo Microsoft Teams
- Visualización de cinta y cabezal en tiempo real
- Ejecución paso a paso o automática
- Control de velocidad de ejecución
- Historial detallado de ejecución
- 10 expresiones regulares predefinidas
- Determinación de aceptación/rechazo

## Instalación y Ejecución

### Requisitos
- Python 3.6+
- tkinter (incluido en instalaciones estándar de Python)

### Ejecución
```bash
# Clonar repositorio
git clone 
cd simulador-maquina-turing

# Ejecutar aplicación
python simulador_turing.py
